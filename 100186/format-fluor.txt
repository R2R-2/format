File Format ID: scs_fluor

Format Description Creation Date: 2019-05-20

File Type: ASCII, comma-separated values

Record Format: Fluorometer output string with external
datetime stamp.

Data Format Description:
The file contains Fluorometer data strings prefixed by an external datetime stamp.

Fluorometer fields are user defined. Example Record:
03/15/2010,14:08:45.876,26


where:
     03/15/2010     Date (source unknown), UTC
                      03   = 2-digit month
                      15   = 2-digit day of month
                      2010 = 4-digit year
     14:08:45.876   Time (source unknown), UTC
                      14     = 2-digit hour [24-hour clock]
                      08     = 2-digit minute
                      45.876 = decimal seconds
     followed by user defined Fluorometer fields
