Data Format Description: These files contain non-contact sea surface "skin" temperature measurements. Each data line includes a log identifier, a high-resolution ISO 8601 UTC timestamp, and the temperature value followed by a literal "C" unit designation.

Standards Followed: ISO 8601 (Timestamping)

Useful References:

Sample Rate: 1 Hz (one sample per second)

Data line column descriptions (one description per unique line format):

Example Line: thermo_pyrometer-ct15    2025-09-06T00:00:00.5129Z     12.36 C
where:
  thermo_pyrometer-ct15  Log Name: Static identifier for the infrared pyrometer log.
  2025-09-06T...         Log Time: ISO 8601 standard UTC timestamp (YYYY-MM-DDTHH:MM:SS.ssssZ).
  12.36 C                Temperature: Sea surface skin temperature in degrees Celsius (Â°C). 
                         Note: The value includes a space and the character "C" as a suffix.
