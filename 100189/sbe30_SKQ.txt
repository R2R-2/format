Data Format Description: These files contain high-accuracy sea surface temperature data from a Sea-Bird digital thermometer. Each line consists of a static log identifier, an ISO 8601 formatted UTC timestamp, and the measured temperature in degrees Celsius.

Standards Followed: ISO 8601 (Timestamping)

Useful References:

Sample Rate: ~1Hz

Data line column descriptions (one description per unique line format):

Example Line: thermo_sbe38_cb    2025-09-07T00:00:00.7890Z    11.8332
where:
  thermo_sbe38_cb      		Log Name: Static identifier for the centerboard temperature log.
  2025-09-07T00:00:00.7890Z	Log Time: ISO 8601 standard UTC timestamp (Year-Month-Day T Hour:Minute:Second.Subsecond Z).
  11.8332              		Temperature: Water temperature measured in degrees Celsius (Â°C) according to the ITS-90 scale.
